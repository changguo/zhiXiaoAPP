# 基于Vue.js重构知晓APP项目

##下载
1. 在空白目录直接clone该项目
```js
  $ git clone git@github.com:chping2125/zhiXiaoAPP.git
```

2. 安装依赖模块
```js
  $ npm install
```

3. 创建本地工作分支
shy分支：
```js
  $ git checkout -b shy
```

lee分支：
```js
  $ git checkout -b lee
```

lxn分支：
```js
  $ git checkout -b lxn
```

chping分支：
```js
  $ git checkout -b chping
```


每个人创建完分支以后与远程分支建立联系
```js
  $ git remote origin git@github.com:chping2125/zhiXiaoAPP.git

  $ git push up--stream
```
